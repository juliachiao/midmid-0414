期中報告心得與理念

【本學期心得】

在這門課程中，我學到了很多關於網頁設計與開發的知識，透過平常的實際操作練習，讓我更加理解如何將一些程式技巧實際應用到網頁設計中。每一個作品都是經過不斷的修改與嘗試後，才能產出滿意的成果。這段過程不僅讓我對設計網頁更加熟悉，也持續累積學習經驗，讓自己更進步。

【設計理念】

本作品以「數位生態系」為核心概念，將抽象的程式碼具象化為生機蓬勃的水族箱。透過毛玻璃質感模擬水底世界的清透與深度，每一株海草與魚兒，都象徵著本學期累積的技術，在數位空間中共同呼吸、成長。

【技術實踐說明】
  • Vertex (頂點座標)： 運用 beginShape() 與頂點座標運算，模擬海草與魚鰭的有機律動，將數學曲線轉化為動態美學。
  • Class (物件導向)： 透過類別封裝海草、氣泡與魚群的屬性與行為，實現數位生態元件的模組化管理。
  • Array (陣列) 與 迴圈： 運用陣列儲存複數動態物件，並透過迴圈高效渲染，營造出豐富且具層次感的水族環境。
  • Vector (向量物理)： 實作向量位移運算，達成氣泡躲避滑鼠以及魚群徐徐遊動的即時交互感。
  • Iframe (內嵌框架)： 建立中央整合展示視窗，達成母網頁導覽系統，無需跳頁即可下載載入各週作業。
  • CSS 佈局與 UX： 運用 absolute 定位與毛玻璃特效，搭配 Active State 選中狀態，打造現代感十足的介面。
